## Description

A First-Person Platformer built in Unreal Engine, inspired by games such as Ghostrunner, ULTRAKILL and Neon White.
The project focuses on high-speed travesal and offers challenging platforming levels that reward players who master its movement mechanics.

## Scope

My aim is to create a tutorial level and at least one full level that requires knowledge of all the mechanics to be completed.

On top of basic movement, the player will be able to:
- Sprint
- Dash
- Wall run
- Slide
- Use a grappling hook

## Learning Goals

- First Person character movement
- Implementation of custom traversal mechanics
- Level design
- Usage of Unreal's Gameplay Ability System
